# Travel Guide Website

Built a responsive TripAdvisor-style travel web application from scratch using HTML and CSS, optimized for desktop and mobile devices. This project features a modern travel guide platform with a clean and user-friendly interface.

## Features

- Responsive website layout
- Custom header with logo and navigation menu
- Search section with destination filters
- Featured destinations section
- Hotel and restaurant listing cards
- Mobile-friendly design using CSS Media Queries
- Built using Flexbox and CSS Grid

## Technologies Used

- HTML5
- CSS3

## Sections Included

- Header
- Navigation Menu
- Search Section
- Featured Destinations
- Popular Hotels
- Top-Rated Restaurants
- Footer

## Objective

To create an original and responsive travel guide website using HTML and CSS without copying existing travel platforms.
